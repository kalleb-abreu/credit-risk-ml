# credit-risk-ml

## Data

Large data files are not tracked in git. Download them manually and place under `data/raw/`.

| Dataset | Source | Notes |
|---|---|---|
| ULB Credit Card Fraud | https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud | Place in `data/raw/ulb-credit-card-fraud-detection/` |
| IEEE-CIS Fraud Detection | https://www.kaggle.com/competitions/ieee-fraud-detection/data | Place in `data/raw/ieee-cis-fraud-detection/` — requires accepting competition rules |

Both datasets are heavily imbalanced.
